# Ananya's Musical Bookstore — Flask Web App

**Name:** Ananya Devraj

## About

A music-themed bookstore web application built with Flask and SQLite. It stores 13 books across 4 categories — Biographies, Learn to Play, Music Theory, and Scores and Charts — and lets you browse by category, search by title, view book details, add new books, and filter books available to read now.

## Run the app

```bash
python3 app.py
```

Then open `http://localhost:8080` in your browser.

## Features

1. Home page — browse all categories
2. Category page — view books in a selected category
3. Search — search books by title
4. Book detail — view full info for a single book
5. Add Book — add a new book to the database
6. Read Now — filter books available to read immediately
